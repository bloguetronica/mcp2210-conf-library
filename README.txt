This initial package contains the default configuration and EEPROM contents
for the MCP2210 device from Microchip. The included files are to be used with
MCP2210 Configurator, and showcase the capabilities of this configuration
application. More configuration files should be added in the future.

A description of the included files follows:
– default.bin – Default EEPROM contents;
– default.xml – Default configuration.

To know more about CP2130 Configurator, and download the application, go to:
– https://github.com/bloguetronica/mcp2210-conf
– https://gitlab.com/bloguetronica/mcp2210-conf
